# Brain-games

### Mind Games is a set of five brainâ€”boosting games. Each game asks three questions. After three correct answers in a row, it is considered that the game is over.

#### Presented games:
- Calculator. Arithmetic expressions that need to be calculated.
- Progression. Search for missing numbers in a sequence of numbers.
- Definition of an even number.
- Finding the Greatest Common Divisor.
- Definition of a prime number.


## Install
```
git clone https://github.com/EgorKurilko/fullstack-javascrit-project-44.git
cd fullstack-javascrit-project-44
make install
npm install
npm link
```
